DeepFlow — Option B Wireframes

Chat always on. Pure layout with data inside each panel. Target: 1200px viewport.

Primary project view. DAG canvas with node selected → detail panel open.

  • Projects (active indicator)

  • Header:

  • Context badge: current project/task name
  • Message list (scrollable):
  • AI messages (left-aligned, grey bg)
  • User messages (right-aligned, brand blue bg)
  • System messages (centered, muted — e.g. "Task updated")
  • Action chips on AI messages: "Apply to task", "Create workflow", "Assign to…"
  • Input bar (bottom, sticky):
  • Text input: "Ask about your workflow…"

Top Bar h:44px
  • Breadcrumbs: All Projects › [Project] › [Workflow]
  • View switcher: Graph | Kanban | List (toggle buttons)

  • Search input: "Search tasks…"
Hub Bar (collapsed) h:36px

  • Workflow name
  • Progress bar + percentage
  • Status counts: ✓ done · ▶ active · ⚠ blocked · ○ pending
  • Health meter (5 segments: green/amber/empty)
  • Due date
Graph Canvas flexible width
  • Background:
  • Dot grid pattern
  • Nodes (draggable):
  • Status dot (coloured by status)
  • Task name
  • Assignee avatar (circle = human, hexagon = AI)
  • Selected node: blue border + glow
  • Edges (between nodes):
  • Dependency lines (solid or dashed)
  • Coloured by upstream status
  • Zones (dashed border regions):
  • Level label (e.g. "L2 — Capital Modelling")
  • Tinted background per level
  • Controls:
  • Zoom +/−/% (bottom-left)
  • Minimap (bottom-right)
Detail Panel 360px
  • Header:
  • Task name
  • Close (✕) button
  • Tabs:
  • Overview | Sub-tasks | Comments | History
  • Overview tab fields:
  • Status badge (coloured pill)
  • Assignee: avatar + name + "AI" tag if agent

  • Checkbox + task name + status label

  • Upstream: list with status dots
  • Downstream: list with status dots
Layout math: 56 + 320 + canvas + 360 = 1200px → canvas = 464px.
Detail panel hidden: canvas expands to 824px. Chat always remains.
Hub bar expand: grows to ~220px, pushes canvas down. Shows donut chart, status breakdown bars, upcoming due dates, top assignees.

Layout Summary — Option B

Fixed panels (always present):
— Sidebar: 56px (icon-only, always collapsed in Option B)
— Chat Panel: 320px (always visible, contextual to current view/task)

Variable panels:
— Top Bar: 44px height, spans remaining width
— Hub Bar: 36px collapsed / ~220px expanded, only in Graph + Active Task views
— Detail Panel: 360px, present in Graph + Active Task views. Overlay in List view. Absent in Workspace/Gallery/Settings

Content area widths (of 1200px total):
— With detail panel: 1200 − 56 − 320 − 360 = 464px canvas
— Without detail panel: 1200 − 56 − 320 = 824px content
— Detail panel closed in graph: 1200 − 56 − 320 = 824px canvas

Chat context per view:
— Workspace: general ("What needs attention?")
— Gallery: project-level ("Create from template")
— Graph: workflow-level ("Show blocked tasks")
— Active Task: task-level ("Summarise stress test results")
— Settings: config-level ("Change notification preferences")
